## 1. 软件简介

知题是一款面向职业资格考试、自学备考人群的智能刷题工具。核心特点：

- **本地离线**：所有题库与学习记录存储在本地，无需联网、无需账号。
- **多模式学习**：浏览学习、答题练习、背题卡片、模拟考试、每日一练、智能复习六种模式。
- **科学复习**：内置 SM-2 间隔重复算法，依据遗忘曲线安排复习。
- **数据可视化**：学习趋势、题型分布、标签弱项诊断一目了然。
- **批量导入**：支持任意符合模板的 Excel 题库一键导入，桌面端更可直接拖入文件。
- **全平台一致体验**：macOS 桌面端、浏览器、手机浏览器（响应式 + PWA）共用同一套题库与逻辑。

## 2. 安装与启动

### 2.1 macOS 桌面端

1. 双击 `知题-1.0.0-arm64.dmg`（Apple Silicon）或对应 Intel 版本。
2. 在弹出的窗口中，将「知题」图标拖入「Applications」文件夹。
3. 从「启动台」或「应用程序」中打开知题。

> **首次打开提示"无法验证开发者"**：这是因为应用未做 Apple 付费签名。解决方法：在「应用程序」中**右键点击知题图标 → 选择"打开" → 再次确认"打开"**。此操作仅需一次。

### 2.2 浏览器 / 移动端 PWA（可选）

在支持的浏览器中访问部署地址，点击地址栏的"安装"图标即可将知题添加到桌面或手机主屏，同样支持离线使用。

## 3. 界面总览

### 3.1 桌面端布局

- **左侧边栏**：品牌区、主导航（首页 / 学习 / 错题本 / 统计 / 题库管理等）、日夜模式与提醒设置。侧边栏可通过中缝按钮收起/展开。

- **主内容区**：当前页面内容。

### 3.2 移动端布局

- **底部标签栏**：首页 | 学习 | 错题 | 我的 + 中央练习按钮（FAB）。
- **"学习""我的"标签**：点击弹出底部菜单，聚合更多入口。
- **右上角菜单按钮**：展开完整侧边抽屉。
- **左划删除**：题库管理页对条目左划即可删除。

## 4. 题库管理

### 4.1 导入题库

**方式一：按钮导入**
1. 进入「题库管理」页，点击右上角「导入题库」。
2. 选择本地 `.xlsx` / `.xls` / `.csv` 文件。
3. 系统自动识别列名并预览前几行，确认无误后正式入库。

**方式二：拖拽导入（桌面端）**
在桌面端任意页面，直接将 Excel / CSV 文件**拖入窗口**，放开后自动开始导入，并弹出遮罩提示进度，完成后跳转到题库管理页。

**推荐 Excel 列名格式：**

| 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 答案 | 解析 | 难度 | 标签 |
|------|------|-------|-------|-------|-------|------|------|------|------|
| 单选题 | …… | …… | …… | …… | …… | A | …… | 适中 | 章节一 |

- **题型**：单选题 / 多选题 / 判断题
- **答案**：单选填 `A`，多选填 `ABD`，判断填 `正确` / `错误`（也支持 `A,B` / `A B` / `A、B` 等多种分隔）
- **标签**：多个标签用逗号分隔，用于后续弱项诊断

> 可在导入界面下载 `template.xlsx` 模板参考。

### 4.2 维护题库

- 查看题库列表（名称、题量、创建时间）。
- 导出题库为 Excel 备份或分享。
- 删除题库：桌面端点击删除按钮；移动端左划条目删除。
- 顶部**下拉刷新**（移动端）：在题库管理页顶部下拉即可重新加载数据。

## 5. 学习与练习

进入「学习」可选择以下模式：

### 5.1 浏览学习
按顺序或随机浏览题目，点击「显示答案」查看解析，可随时收藏或添加笔记。支持**左右滑动切题**（移动端）。

### 5.2 答题练习
选择题库、设定题量，逐题作答。支持**选项乱序**（防止死记位置）与标签/难度筛选。提交后即时判题并展示解析。答题页底部提供固定的「上一题 / 提交答案 / 下一题」操作栏。

- **专注模式**：点击答题页右上角「专注模式」按钮，隐藏侧栏与底栏，扩大答题区，沉浸刷题（再次点击退出）。
- **失焦自动暂停**：切到其他 App 或切到后台时，单题计时自动暂停，回到知题继续计时，避免计时失真。
- **Dock 进度条（桌面端）**：答题过程中，macOS 程序坞中的知题图标会显示当前答题进度。
- **左右滑切题（移动端）**：在答题卡片上左右滑动即可切换上一题 / 下一题。
- **触感反馈（移动端）**：提交答案后，答对 / 答错会触发不同的震动提示。

### 5.3 背题模式
卡片式 3D 翻转：正面题干，翻转看答案，适合快速记忆。

### 5.4 模拟考试
设定题量与时长，倒计时进行；超时自动交卷。交卷后生成成绩单（总分、正确率、各题型得分、用时），支持错题回顾。考试期间同样具备「失焦暂停计时」与「Dock 进度条」（桌面端）。

### 5.5 每日一练
每天自动生成一组练习题，含固定底部操作栏，帮助养成打卡习惯。

### 5.6 智能复习
基于 SM-2 间隔重复算法，优先推送到期、错误次数多的题目。作答后根据掌握度评分（记得清楚 / 模糊 / 忘记）自动安排下次复习时间。

## 6. 错题本与收藏

### 6.1 错题本
- 自动归集所有答错题目，按错误次数降序排列。
- 支持一键重新练习。

## 10. 移动端专属体验

在手机浏览器中使用知题，除响应式布局外，还针对触屏做了专属优化：

- **横屏布局**：手机横屏时，题目选项自动改为两列网格、卡片限宽，避免内容过窄。
- **虚拟键盘避让**：输入框（搜索、备注）聚焦、软键盘弹出时，底部导航与答题底栏自动上移，避免被遮挡。
- **左右滑切题**：练习 / 考试 / 浏览页左右滑动切换上一题 / 下一题。
- **触感反馈**：答题对错触发不同震动（支持震动的设备）。
- **下拉刷新**：题库管理页顶部下拉刷新数据。
- **刘海安全区适配**：顶部状态栏、底部 Home 指示条区域自动留白。

## 11. 数据安全与备份

- **本地存储**：桌面端数据库文件保存在系统用户数据目录（经主进程 IPC 落盘，非浏览器 OPFS）；浏览器 / PWA 端使用 OPFS 持久化。
- **完全离线**：无网络依赖，数据不上传。
- **备份建议**：定期通过「题库管理 → 导出」保存 Excel 备份。

## 12. 数据同步（零服务器、零费用）

不想花钱、没有服务器，也能在多台设备间同步学习进度。知题提供两种后端，在「数据同步」弹窗顶部切换：

- **GitHub 私有仓库**：复用 GitHub 账号，进度存到一个私有 Gist。
- **坚果云 WebDAV**：复用坚果云账号，进度存到坚果云的同步目录（**国内访问更稳更快**）。

> **平台限制（重要）**：纯前端从手机网页（PWA）直连云存储会被浏览器 CORS 拦截。GitHub 的 API 主动返回了 CORS 头，所以手机网页也能用；坚果云 WebDAV 不返回 CORS 头，因此**坚果云仅支持桌面端（Mac 版）**，手机网页版不可用（弹窗内会提示）。

### 方式一：GitHub 私有仓库

#### 准备工作
1. 打开 GitHub → Settings → Developer settings → Personal access tokens。
2. 生成一个 Token，**只需勾选 `gist` 作用域**（Fine-grained 令牌则授予 Gist 读写权限）。
3. 复制生成的 Token（`ghp_` 或 `github_pat_` 开头）。

#### 使用步骤
1. 侧边栏底部点击「数据同步」，选择「GitHub 私有仓库」，粘贴 Token 并保存（Token 仅存本机，不上传第三方）。
2. 点击「立即同步」：先把远端进度合并进本地，再把合并结果上传。
3. 可勾选「打开应用时自动同步」，下次启动自动拉取合并。

### 方式二：坚果云 WebDAV（国内推荐，桌面端）

#### 准备工作
1. 注册/登录坚果云（jianguoyun.com）。
2. 网页端「设置 → 安全选项 → 添加应用」，生成一个**应用密码**（注意：是应用密码，不是登录密码，勾选「读写」权限）。
3. 记下服务器地址（默认 `https://dav.jianguoyun.com/dav`，一般无需改动）。

#### 使用步骤
1. 侧边栏底部点击「数据同步」，选择「坚果云 WebDAV」，填写服务器地址、账号（邮箱）、应用密码，点「保存配置」。
2. 点击「立即同步」：进度会写入坚果云 `知题/zetith-sync.json`，多台桌面端共用同一账号即可同步。
3. 可勾选「打开应用时自动同步」。

### 同步范围与规则
- 同步的是**学习进度**（SM-2 复习状态 / 笔记 / 收藏），**不包含题库本身**（题库体积大，各端本地导入即可）。
- 进度以题目 ID 为键，请在各设备导入**相同的题库文件**，题目 ID 才能正确对应。
- 冲突以「最后修改时间」为准，多端同时改动时较新的一方生效；不会删除本地独有记录。

## 14. 常见问题

**Q：打开桌面端提示"应用已损坏"或"无法验证开发者"？**
A：应用未做付费签名。右键图标 → 打开 → 确认即可，仅需一次。

**Q：导入题库后题目数量不对？**
A：请检查 Excel 是否有空行、必填列（题型/题干/答案）是否缺失，系统会跳过无效行。也可直接将文件拖入窗口导入。

**Q：数据会丢失吗？**
A：数据存于本地，卸载应用或清理浏览器数据可能导致丢失，建议定期导出备份。

**Q：深色模式为什么有时不跟随系统？**
A：若你曾手动按 `⌘T` 切换过主题，知题会改用你手动设定的主题；想恢复跟随系统，请点击菜单「视图 › 跟随系统外观」。

**Q：支持 Windows / Linux 吗？**
A：当前打包版本为 macOS；技术栈基于 Electron，可扩展支持 Windows / Linux。
